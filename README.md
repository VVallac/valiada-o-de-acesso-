# Validador de Acesso

Uma solução de validação de acesso com bloqueio automático de IPs suspeitos.

## Descrição

Este projeto implementa uma lógica de validação de acesso em Python com proteção contra tentativas de acesso não autorizado. O sistema monitora tentativas de login falhadas e bloqueia automaticamente IPs após um determinado número de erros.

## Características

- Contador de erros por IP
- Bloqueio automático após 5 tentativas falhadas
- Alerta de IP suspeito
- Integração com tabela de IPs do servidor
- Bloqueio persistente de IPs maliciosos
